# About
this design pattern emphasis making your project in terms of:
1. atomic components: is the most basic component that could be even just a normal button
2. molecules components: is a group of atomic components, it could represent a form or a simple component
3. organisms components: is a group of molecules components, it could represent a large component or more complex component than molecules
4. template components: is a group of organisms components, it could represent it could represent boilerplate components for all pages of the application or some of them
5. page components: the actual page as a whole


# Note
Usually in this pattern each component takes a props to be dynamically rendered as needed.
