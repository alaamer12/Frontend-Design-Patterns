# About
this design pattern emphasis making your project in terms of:
1. injected component: it is a component which rendered as children based on some constraints or conditions, also the injected component could be another injected component, it is injected into the injector[example PluginHost]
2. injector component: are the components that are responsible for rendering the injected component

# Note
there are different types of injection we only demonstrated the most common one
