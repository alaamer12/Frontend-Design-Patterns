# About
this design pattern emphasis making your project in terms of:
1. controlled component: it is a component that has its own state, and its state is handled by its parent component
2. uncontrolled component: or external component or parent component, it has different names, it is a component that does not have its own state, but it controls or handles the state of another component


# Note
there is different types of this pattern, some of them could update the parent state based on the child state
